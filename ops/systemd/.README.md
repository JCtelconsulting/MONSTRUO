# Systemd Services
Archivos de servicio para systemd.
## Estructura Permitida (Allowlist)
La IA NO debe crear archivos fuera de esta lista sin permiso.
- *.service
- *.timer

## Nota operativa
- En este repositorio (rama `dev`) el runtime principal es Docker Compose.
- Estas unidades systemd se mantienen como referencia/legacy y deben validar rutas antes de usarse.
