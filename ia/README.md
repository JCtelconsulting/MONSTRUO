# IA — Asistente Inteligente (ULTRON)

Módulo de integración con modelos de lenguaje para soporte interno. Procesa consultas, analiza datos y asiste a los equipos de la plataforma.

## Estructura

```
ia/
├── backend/       # API FastAPI y lógica de integración LLM
├── ui/            # Frontend del asistente
├── migrations/    # Migraciones SQL del schema ia.*
├── tests/         # Tests de integración
├── scripts/       # Scripts de utilidad
├── docs/          # Documentación del módulo
└── data/          # Datos locales (excluidos de git)
```
