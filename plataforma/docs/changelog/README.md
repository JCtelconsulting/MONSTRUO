## Convención

- Un archivo por mes (`YYYY-MM.md`).
- Cada hito como `## HITO: YYYY-MM-DD - <título corto>`.
- El más nuevo va arriba dentro del mes.
- Hitos del mes en curso se agregan acá; el primer día del mes siguiente se crea el archivo nuevo.
