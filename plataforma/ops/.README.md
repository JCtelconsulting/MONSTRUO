# Ops
Operaciones, herramientas y servicios del sistema.
## Estructura Permitida (Allowlist)
La IA NO debe crear archivos fuera de esta lista sin permiso.
- compose/
- control/
- entornos/
- systemd/
- guardian/
- herramientas/

## Reglas
- No guardar secretos reales en scripts (`SUDO_PASS`, tokens, claves).
- Usar variables de entorno o `.secrets/` fuera de Git.
- Mantener separación DEV/PROD en rutas y nombres de entorno.
