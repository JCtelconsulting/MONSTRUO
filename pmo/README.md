# PMO — Oficina de Gestión de Proyectos

Módulo de gestión de proyectos técnicos e implementaciones. Seguimiento de hitos, recursos y entregables por proyecto.

## Estructura

```
pmo/
├── backend/       # API FastAPI y lógica de proyectos
├── ui/            # Frontend de gestión de proyectos
├── migrations/    # Migraciones SQL del schema pmo.*
├── tests/         # Tests de integración
├── scripts/       # Scripts de utilidad
├── docs/          # Documentación del módulo
└── data/          # Datos locales (excluidos de git)
```
