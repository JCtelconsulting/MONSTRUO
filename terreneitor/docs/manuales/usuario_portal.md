# Manual de Usuario - Portal Administrativo

El Portal es el centro de mando técnico y administrativo de Terreneitor.

## 1. Accesos Directos
Desde la pantalla de inicio del Portal, puede saltar rápidamente a los módulos de Terreno, Supervisión o Gerencia.

## 2. Monitor de Sistema
La pestaña **"Administración"** muestra la salud del servidor:
- Uso de CPU y RAM.
- Estado de la conexión con Google Drive (Backup).
- Temperatura del servidor (prevención de fallos de hardware).

## 3. Gestión de Usuarios
En la pestaña **"Usuarios"**:
- **Crear Usuario**: Asignar Email, Nombre, Clave y Rol.
- **Roles**:
    - `ADMIN`: Acceso total.
    - `GERENCIA`: Dashboard ejecutivo.
    - `SUPERVISOR`: Gestión de terreno.
    - `TERRENO`: Ejecución en campo.
- **Resetear Claves**: El administrador puede cambiar la clave de cualquier usuario si es necesario.

## 4. Gestión de Proyectos
En la pestaña **"Proyectos"**:
- **Nuevo Proyecto**: Crear la base para la operación.
- **Plantillas**: Seleccionar el tipo de trabajo (PMC, SATLINK, OBRA, DOMICILIO) para que el sistema genere automáticamente las carpetas y tareas necesarias.
- **Editor de Estructura**: Permite mover fotos entre carpetas, eliminar ítems o agregar nuevas tareas personalizadas a un proyecto existente.

## 5. Mantenimiento (Sistema)
- **Sincronizar Archivos**: Fuerza al sistema a leer el disco duro para detectar nuevas fotos que no hayan sido registradas.
- **Crear Estructura**: Repara carpetas faltantes en el servidor.
