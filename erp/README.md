# Módulo: ERP & Finanzas 💰

## Propósito
Gestión administrativa y financiera: Facturación (DTE), Compras, Conciliación Bancaria y sincronización con Laudus.

## Estructura Local
- `main.py`: API del módulo.
- `laudus.py`: Integración con API de Laudus.
- `sales_service.py`: Lógica de ventas y facturación.
- `ui/`: Vistas de finanzas y conciliación.
