# Procesos de Negocio
Scripts python que ejecutan lógica de negocio fuera del ciclo HTTP.
- **integracion/**: Jobs programados de sincronización.
- **mantenimiento/**: Scripts de corrección de datos.
- **ai/**: Workers de inteligencia artificial (async).

## Estructura Permitida (Allowlist)
La IA NO debe crear archivos fuera de esta lista sin permiso.
- ai/
- integracion/
- mantenimiento/
