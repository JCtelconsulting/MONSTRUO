# Frontend Static
Archivos servidos estáticamente por FastAPI (SPA/MPA).
- **modulos/**: Funcionalidades completas (Login, Dashboard, Bodega, etc).
  - **_compartido/**: Recursos globales (CSS base, JS utils, Img).
## Estructura Permitida (Allowlist)
La IA NO debe crear archivos fuera de esta lista sin permiso.
- modulos/
- manifest.json
- service-worker.js
- index.html
