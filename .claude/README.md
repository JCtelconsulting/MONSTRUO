# .claude/

Carpeta para configuración específica de Claude Code en este repo.

Por ahora vacía — reservada para futuros subagentes (`.claude/agents/<nombre>.md`)
o configuración de hooks (`.claude/settings.json`).

Las **reglas operativas** del repo (válidas para cualquier agente, no solo Claude)
viven en `AGENTS.md` en la raíz. `CLAUDE.md` apunta a ellas.
